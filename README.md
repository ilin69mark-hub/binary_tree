# Двоичное дерево на языке Go

## Описание
Эта программа реализует структуру данных "двоичное дерево" с основными операциями:
- Вставка элемента
- Удаление элемента
- Поиск элемента
- Печать элементов дерева

## Структура программы

### TreeNode
Структура узла двоичного дерева:
```go
type TreeNode struct {
    Value int
    Left  *TreeNode
    Right *TreeNode
}
```

### BinaryTree
Структура двоичного дерева:
```go
type BinaryTree struct {
    Root *TreeNode
}
```

## Функции

### Вставка элемента
```go
func (bt *BinaryTree) Insert(value int)
```
Добавляет новый элемент в дерево по правилам бинарного поиска: 
- Если значение меньше текущего узла, оно добавляется в левое поддерево
- Если значение больше текущего узла, оно добавляется в правое поддерево

### Поиск элемента
```go
func (bt *BinaryTree) Search(value int) bool
```
Возвращает true, если элемент найден в дереве, иначе false.

### Удаление элемента
```go
func (bt *BinaryTree) Delete(value int)
```
Удаляет элемент из дерева. Обрабатывает три случая:
- Удаляемый узел не имеет потомков
- Удаляемый узел имеет одного потомка
- Удаляемый узел имеет двух потомков

### Печать элементов
Программа предоставляет три способа обхода и печати элементов:
- In-order (левое поддерево, корень, правое поддерево)
- Pre-order (корень, левое поддерево, правое поддерево)
- Post-order (левое поддерево, правое поддерево, корень)

## Как запустить программу

1. Убедитесь, что у вас установлен Go (версия 1.16 или выше)
2. Склонируйте или создайте файл `main.go` с кодом программы
3. Выполните команду в терминале:

```bash
cd /workspace
go run main.go
```

Или вы можете скомпилировать программу и затем запустить:

```bash
go build -o binary_tree main.go
./binary_tree
```

## Пример вывода
При запуске программы вы увидите:
- Последовательное добавление элементов в дерево
- Три различных способа печати элементов
- Результаты поиска заданных элементов
- Последовательное удаление элементов с отображением изменений
